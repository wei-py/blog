2025-04-27    gitcommitlintcz-gitcommitizen

git commmit 提交规范 ​

安装 ​

shell
npm install -g commitizen
pnpm install -D cz-git

package.json ​

json
{
  "scripts": {
    "commit": "git-cz"
  },
  "config": {
    "commitizen": {
      "path": "node_modules/cz-git",
    }
  }
}

commitlint.config.js ​

js
